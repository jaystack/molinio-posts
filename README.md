# molinio-posts

## Manual

### file structure

molinio-posts
|
|- post-folder/ (use unix valid folder name)
|   |
|   |- post.md (Markdown file)
|   |- meta.conf (Post meta data)
|   |- post-image-1.jpg
|   |- post-image-2.png

### meta.conf content

    title="Post title"
    slug="/post-title"
    author="example@example.com"
    category="Category1,Category2"
    tag="Tag1,Tag2"
    date="2016-12-24 07:00:00"
